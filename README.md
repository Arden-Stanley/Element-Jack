# Element-Jack

---

## Introduction

Implementation of the game BlackJack with a custom language known as ElementLang.

---

## How to Run

### MacOS, Linux, WSL

```bash
chmod +x build.sh
chmod +x rmcmt.sh
./rmctm.sh src/Blackjack.elem <output file>
./build.sh # you only need to run this if you have already used the lines above
```

### Windows

```powershell
.\run.ps1
```
