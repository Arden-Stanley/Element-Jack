# Element-Jack

---

## Introduction

Implementation of the game BlackJack with a custom language known as ElementLang.

---

## How to Run

### MacOS, Linux, WSL

```bash
chmod +x run.sh
chmod +x rmcmt.sh
./run.sh # you only need to run this if you have already used the lines above
```

### Windows

```powershell
.\run.ps1
```
